Tampermonkey Script (One-Click Save)

Install the Tampermonkey userscript to add a save button directly to every tweet on Twitter/X — no need to copy links manually.

  1. Install the Tampermonkey browser extension Available for Chrome, Firefox, Edge, and Safari
  2. Make sure this service is running, then Click to install script Tampermonkey will open an installation confirmation page
  3. Click Install on the Tampermonkey confirmation page
The script file is at tampermonkey/twitter-saver.user.js. You can also copy its contents manually into a new Tampermonkey script.
Configure Backend URL

The script connects to http://localhost:6201 by default. If your service runs at a different address, update the setting:

  1. Click the Tampermonkey icon in your browser toolbar
  2. Find the Twitter/X Saver Save Button script
  3. Click ⚙️ Set Backend URL in the script menu
  4. Enter your service address, for example:
    • http://localhost:6201 (local)
    • https://your-server.com (remote server)
  5. Click OK to save, then refresh Twitter/X for it to take effect
Usage
  1. Open Twitter/X
  2. A blue save button appears at the right end of every tweet's action bar
  3. Click it — the script submits the tweet URL to your local archiver
  4. A toast notification appears in the top-right:
    • Green — Task submitted successfully
    • Yellow — Tweet already saved
    • Red — Submission failed — check that the service is running
  5. Check progress in the Task Queue, then browse results in Saved
FAQ

  • Make sure Tampermonkey is enabled and the script is active
  • Refresh Twitter/X and wait 1–2 seconds for the script to initialize
  • Check the browser console (F12) for error messages
  • Try disabling other Twitter-related scripts to rule out conflicts

The icon is embedded in the script itself. If it still fails, try reinstalling the script from the install link.

  • Make sure the service is running: python run_web.py
  • Check and update the backend URL via ⚙️ Set Backend URL
  • If using a remote server, confirm the port is open in your firewall
  • In Tampermonkey, confirm @connect * is granted

Revisit the install link — Tampermonkey will prompt you to overwrite. Or manually replace the script content in the Tampermonkey dashboard.
